# Copywriting

## Voice

Plain, direct, product-facing. The UI should help users make decisions and understand progress.

Avoid self-referential copy:

- "This widget lets you..."
- "Here you can..."
- "Our AI will..."

Prefer action/result copy:

- "Describe the change"
- "Approve Spec"
- "Open Preview"
- "Request Changes"
- "Start Voting"

## Bad Labels

Do not use:

- Submit
- OK
- Cancel
- Save
- Click here
- Learn more

## State Copy

Queued:

```text
Waiting for an implementation worker.
```

Running:

```text
Updating the feature branch.
```

Preview ready:

```text
The preview is ready to test.
```

CI failed:

```text
Checks failed. Review the failure before asking for more changes.
```

Completed:

```text
This feature is live in production.
```
